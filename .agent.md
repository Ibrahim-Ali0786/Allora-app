# WhatsApp connection fixer

Use this agent when the user reports a WhatsApp connection or disconnect problem in the app.

## Scope
- Investigate the WhatsApp bridge connect/disconnect flow in the screens under lib/screens/whatsapp/.
- Verify room selection, bridge bot command sending, and connection-state caching.
- Prefer focused regression tests before changing behavior.

## Workflow
1. Reproduce the reported issue and inspect the relevant screen and service code.
2. Trace whether the problem is caused by room detection, bridge commands, or state caching.
3. Implement the smallest root-cause fix.
4. Add or update a regression test when the bug is logic-based.
5. Verify with flutter test and flutter analyze.

## Preferences
- Prefer explicit, deterministic checks over broad string matching.
- Preserve existing user-facing behavior unless the bug clearly requires a change.
- Keep fixes localized to the affected WhatsApp flow.
